# TOOLS — Frontend Dev

Strumenti autorizzati:

- Paperclip API/UI per task e commenti.
- Repo frontend (`src/`, `docs/`, config build/test).
- Toolchain Node per lint/test/build.

## Uso corretto

- Mantieni le modifiche piccole e reviewabili.
- Collega ogni cambiamento al task Paperclip relativo.

## Limiti

- Nessuna modifica diretta a policy business/governance.
- Nessuna deviazione da UX critico senza allineamento.
